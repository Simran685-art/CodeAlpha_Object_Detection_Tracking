# CodeAlpha_Object_Detection_Tracking
A real-time Object Detection and Tracking System
# 🎯 CodeAlpha Internship Project  
## Object Detection + Tracking using YOLOv8

This project is developed as part of the **CodeAlpha Internship Program**.  
It demonstrates real-time **Object Detection and Tracking** using **YOLOv8 (Ultralytics)** with a Streamlit web application.

---

## 🚀 Features

- 📷 Image Object Detection
- 🎥 Video Object Detection + Tracking
- 📡 Live Webcam Tracking
- 📊 FPS Counter (real-time performance)
- 🔢 Object Counter (person, car, etc.)
- ⬇️ Download processed video
- 🧠 YOLOv8 pretrained model (yolov8n)
- 🎯 ByteTrack tracking algorithm

---

## 🧠 Technologies Used

- Python 🐍
- Streamlit 🌐
- YOLOv8 (Ultralytics)
- OpenCV 👁️
- Pillow 🖼️
